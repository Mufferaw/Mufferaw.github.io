Written in Javascript and built on React Native, this mobile application is a hybrid between audiobook and ebook, allowing users to listen and follow along with the text simultaneously as each word is highlighted as it's being read.

C++ / JUCE Framework Otodecks

A music mixing/DJ application written in C++ and using the JUCE Framework. It features custom GUI controls for adjusting the playback speed and volume of multiple tracks simultaneously.

Javascript - Music Visualizer

This Music Visualizer is written in javascript using the P5 library and features serveal visualizations which can be selected and adjusted using a GUI (drawn in P5 using an Immediate Mode GUI technique). The Rapidly-Exploring Random Tree visualization produces an especially interesting effect when driven with information from the audio being played. Try it out!

Javascript - Audio Mixer

The application is written in javascript using the P5 library as well as the P5.Sound library. The GUI is also drawn using P5 using an IMGUI (Immediate Mode GUI) technique. The application features audio effects such as low/high pass, reverb, compressor and more. Accepts input from audio file or microphone and can export the processed audio. Try it out!

Unity - Adventure Puzzle game

This team project involved the development of a third-person adventure puzzle game. Players control a character who must journey through a treacherous land inhabited by deadly creatures. The goal is to collect misplaced orbs scattered throughout the environment and return them to their respective receptacles. Players must avoid the dangerous creatures and restore the orbs to their rightful places before time runs out. Try it out!

Unity - Space Shooter

A first-person, arcade style, sci-fi themed shoot’em up game for purely entertainment purposes. The player takes control of a spacecraft with the ability to move on the x and y axes while it travels along a ‘rail/path’ in the z-axis. The player must dodge obstacles, enemies and their projectiles while also destroying enemies by using the mouse to aim and shoot. Try it out!

Unity - Pinball

A fun pinball game which uses Unity and it's physics system to provide a realistic and challenging diversion. Try it out!

Unity - Boid Temple

A robotic character moves about the environment animated by simple rotations which, when taken in aggregate, give the character a sense of weight and inertia. Also found in the environment are a 'flock of boids', based on the classic ‘Flocking’ algorithm. The 'boids' consider alignment, separation and cohesion with respect to one another when determining their movement. Try it out!

ejs / express / mysql - Smarthome

A dynamic web application that functions as a dashboard, enabling users to perform CRUD operations for managing devices in their smart home setup. The application, built with EJS, Express, and MySQL, allows users to add new devices from a predefined list (Create), check the status of their existing devices (Read), change the settings of their devices (Update), and remove unwanted devices (Delete).

Javascript - Music Viz with Meida

This webapp uses the Meyda library to extract audio features from a song and then uses that data to drive the colorful visualizations. It uses p5.js for drawing the visualizations and for Web Speech API functionality (Recognition). Users can issue voice commands to choose different shapes and colors for the visualizations. Try it out!

This is a very basic ray tracer written as an exercise to learn about The Red Programming Language. The main part of the ray tracer is written in Red/System, the interface and scene parsing are written in Red. Based on Peter Shirley's "Raytracing in One Weekend" Check it out on Github.
